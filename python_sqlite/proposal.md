# Proposal: Simple Blog System with SQLite

## Objective
Build a lightweight blog application where users can register, log in, and create posts. Only authenticated users may publish new blog entries.

## Scope
- User registration
- User login and logout
- Protected post creation
- View blog posts publicly
- Store data in SQLite

## Proposed Stack
- Python
- Flask (simple web framework)
- SQLite for data persistence
- HTML/CSS/JS for the frontend

## Success Criteria
- Users can create an account
- Users can sign in securely
- Logged-in users can create posts
- Guests can read posts without posting rights
- Data is stored in a SQLite database
